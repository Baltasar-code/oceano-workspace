# Oceano Workspace

Plataforma de espacio de trabajo corporativo diseñada para cPanel.

## Tecnologías
- PHP Nativo (PDO)
- MySQL
- Vanilla JavaScript
- Tailwind CSS

## Instalación
1. Importa `database.sql` en phpMyAdmin.
2. Configura `config/config.php` con tus credenciales.
3. Sube los archivos a tu servidor.
